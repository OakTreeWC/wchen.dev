# wchen.dev
My Personal Website that runs on Flask

To deploy, clone the repo,
'''git clone https://github.com/OakTreeWC/wchen.dev.git'''
cd into it,
'''cd wchen.dev'''
install Flask,
'''python3 -m pip install flask'''
and run!
'''python3 main.py'''
*may need to run with elevated permissions*
