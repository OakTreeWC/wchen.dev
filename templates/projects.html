12KEMP V2.0

Canceled

The 12KEMP V2.0 is a completely revamped version of the original 12KEMP, with a new design, custom case, USB-C, a 0.91" OLED display, and addressable RGB LEDs. Instead of the Arduino Pro Micro, this uses a soldered on Atmega32U4, which allows for a more compact design and more IO pins. Due to some oversight, this version uses 0402 (1005 metric) smd components, instead of the larger 0603 (1608 metric) variant. This does slightly increase assembly difficulty, but not as much as I had feared.
